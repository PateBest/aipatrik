GitHub Secrets Status

Tämä sivu näyttää GitHub Secretsien tilan.

SUPABASE_URL: Määritelty (alkaa: https://og...)
SUPABASE_ANON_KEY: Määritelty (alkaa: eyJhbGciOi...)

Ohjeet

Jos GitHub Secrets puuttuvat, seuraa näitä ohjeita:

  1. Mene GitHub-repositoriosi asetuksiin
  2. Valitse vasemmasta sivupalkista 'Secrets and variables' ja sitten 'Actions'
  3. Klikkaa 'New repository secret' -painiketta
  4. Lisää SUPABASE_URL ja SUPABASE_ANON_KEY secretit Supabase-projektistasi
